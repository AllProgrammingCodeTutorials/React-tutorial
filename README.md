# React-tutorial

## requirement
* node.js: https://nodejs.org/en/
* npmjs: https://www.npmjs.com/
* text Editor : VS Code(highly recommended) : https://code.visualstudio.com/ , Sublime Editor, Notepad++

## Installation
----------------
The project can be installed using command line :

* npx create-react-app my-app
* cd my-app
* npm start
